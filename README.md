## DESCRIPTION

Website designed for the collaboration of students and businesses. A place to advertise your abilities, and request the work of others.Thereby lowering the barrier to professional collaboration

## ARCHITECTURE

## FEATURES

- User Accounts/Authentication: Users can register and log in to create a persistent identity on the platform.
- Project Board: A central browseable listing of projects where users can view, search, and filter available posts.
- Project Creation: Authenticated users can create new project listings directly from the project board.
- Project Roles/Requests: Project owners can request roles. Other users can accept or decline these requests as well as request other roles.
- Responsive Design: The UI is styled to be intuitive, user-friendly, and designed for both desktop and mobile browsing.
